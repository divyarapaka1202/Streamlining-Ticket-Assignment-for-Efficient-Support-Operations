# Streamlining Ticket Assignment for Efficient Support Operations

## Project Overview

This project automates ticket assignment using ServiceNow.
Tickets are automatically assigned to the appropriate support group based on the selected issue type. The system also ensures secure access using role-based control.

## Objective

* Automate ticket routing
* Reduce manual work
* Improve efficiency
* Implement role-based access control

## Technology Used

* ServiceNow
* Flow Designer
* Access Control Lists (ACL)
* Custom Table (Operations Related)
